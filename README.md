##  Project Overview
This project is a high-performance, cross-platform inventory management solution designed to handle complex business logistics. It demonstrates a professional approach to software architecture, combining mobile accessibility with a robust, scalable backend.

### Key Objectives:
* **Business Logic:** Managing real-time stock levels, user authentication, and transaction history.
* [cite_start]**Hybrid Data Architecture:** Utilizing both **SQL** for relational integrity [cite: 24] [cite_start]and **NoSQL (MongoDB)** for flexible logging and metadata[cite: 25].
* [cite_start]**Clean Code:** Implementing **SOLID principles** and **Design Patterns** using Java[cite: 14].
* [cite_start]**International Communication:** Technical documentation and codebase maintained in English (C1 level)[cite: 7, 40].
